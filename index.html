If you would like to center align a element horizontally with respect to the parent element you can use the CSS margin property with the value auto for the left and right side, i.e. set the style rule margin: 0 auto; for the div element. The auto value automatically adjusts the left and right margin and horizontally center align the div element's block box. However, this solution will work only when the width of the parent element is more than the containing div element that you want to align.
